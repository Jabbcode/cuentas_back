# Workflow - controller-generator-agent

Pasos que ejecuta este agent.

## Pasos

1. **Input:** Recibe instrucción
2. **Análisis:** Entiende el requerimiento
3. **Implementación:** Crea la solución
4. **Validación:** Verifica TypeScript y patrones
5. **Seguridad:** Valida userId filtering
6. **Output:** Retorna código listo

## Entradas Esperadas

- Descripción clara de qué crear
- Contexto del proyecto (si aplica)
- Patrones a seguir
- Constraints técnicos

## Salidas Esperadas

- Código funcional
- TypeScript correcto
- Siguiendo patrones
- UserId filtering incluido
- Listo para usar

## Validaciones Críticas

- ✅ TypeScript types correctos
- ✅ Convenciones seguidas
- ✅ Patrones respetados
- ✅ Errores manejados
- ✅ userId filtering presente
